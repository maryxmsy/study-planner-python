# 📚 Study Planner (Python)

A command-line Study Planner built using Python.

## ✨ Features

- Add Tasks
- View Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Search Tasks
- Filter Tasks
- View Statistics
- Export Tasks to a Text Report
- Save Data using JSON

## 🛠 Technologies Used

- Python
- JSON
- File Handling
- Datetime Module

## 🚀 How to Run

1. Download the project.
2. Open the project in Python or Pydroid 3.
3. Run:

```bash
python study_planner.py
```
